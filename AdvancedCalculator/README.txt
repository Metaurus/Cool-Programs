ADVANCED CALCULATOR

Language: Java

STATUS: Complete and Runnable

Last Updated: 2022-4-18

AdvancedCalculator.java is a program that, when run, takes String inputs of equations that can be parsed and calculated. The equations can consist of most arithmetic operators and functions. This program was taken from an online job interview for an internship at a software company (its name will be kept private).

This program requires the user to have Java installed on their system.

Available operations: "+", "-", "*", "/", "^", "!", "mod", "lg", "ln", "cos", "sin", "tan", "sqrt"

Example input: lg(15) + ln(e) - pi / 35
